Git 课程学习总结

学习收获
通过本课程，我基本掌握了 Git 的基本概念和操作，包括版本控制、分支管理、合并操作以及 Pull Request 协作流程。

实践体会
最大的收获是能够使用Git团队协作的工作流程。通过实际创建分支、提交代码、解决冲突和发起 Pull Request，我体会到了版本控制工具的妙处。

未来应用
我计划在未来的个人和团队项目中使用 Git，能更为熟练的使用Git，为以后团队开发打好基础。
