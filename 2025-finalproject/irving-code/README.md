# 学习总结

学习方式

* 听培训课
* 大致浏览一遍作业，大致总结出思路
* 发现实际操作或者构建思路时不懂的点，记录并分类
* 辨别出到底是具体操作不懂还是连思路都没有，如果具体操作不懂，看是因为知识点听都没听过还是有印象但是不熟，如果是思路都没有，考虑将题目交给ai进行思路的分析
* 回看课程，找到在作业应用中，上课讲了没懂和上课完全没懂的部分
* 针对不懂的部分找对应的课程来听，根据自己的理解复述给ai，进行纠偏
* 开始动手写，遇到不懂的重复上面的流程
* 最后实在不懂的地方看下ai的答案，弄明白答案后在答案基础上根据实际情况优化
* 如果写出来了，将答案交给ai看看它能不能给出更好的解决方案，有选择的修改自己的答案。
  * 在以上流程的同时先将大致的知识体系建立起来，然后针对自己的难点痛点记录笔记

收获：

* git基本指令
* bash脚本语言的学习
* 终端命令行
* cmake：通过学习cmake对cpp的编译等过程有了更加全面深刻的认知
