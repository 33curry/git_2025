#### 关于学习路径

在以上内容的学习中，我认为自学占了其中很大的一部分。在摸索中我大致掌握了对于陌生的新知识的学习路径。

1. 优先查阅参考文档与教材

   这些内容通常较为权威，信息全面，缺点是有一定的门槛

   *例如学习**inux命令行**可以使用**man**具体了解命令的功能，或是学习c，c++等编程语言有**c primer plus**等教材作参考*

2. 使用浏览器针对关键字进行查询

   可以查询到包括第一点在内的各种文章或视频教程等，但是需要对内容进行筛选，确保找到的是自己真正需要的内容

   *像是**菜鸟教程**，**csdn**，或是一些视频都是学习的相对门槛较低的途径*

3. 询问ai

   ai是强大的工具，但是需要我们**合理**地提问才能高效地解决问题

   *很多时候ai的回答仅供参考，不是无脑的将问题扔给ai让它帮我们解决问题。ai能显示它思考问题的方式，为我们提供一条**解决问题的路径**，在根据路径运用以上手段的过程中就能逐步学会如何解决问题*

4. 寻求帮助

   请教他人通常是很有用的方法，有经验的人给予的指引通常能带来启发

   *闭门造车不利于学习新知识，多交流也是学习的一部分，会问问题也是一种很重要的能力*
