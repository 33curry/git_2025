# 课程学习总结

## 一、整体感受
这门课让我对编程工具和协作流程有了全新的认识，尤其是 Git 和命令行的使用，从一开始的陌生到现在能独立完成基本操作，进步很明显。

## 二、核心知识点回顾
1. **Git 版本控制**：
   - 掌握了 `git add` `git commit` `git push` 等基础命令。
   - 理解了分支的作用，学会用 `git branch` `git checkout` 管理分支。
   - 解决冲突时，`git merge` 和 `git rebase` 的区别让我花了点时间理解，后来通过实际操作终于理清了。

2. **命令行工具**：
   - 学会用 `cd` `ls` `mkdir` 等命令高效管理文件，比图形界面快很多。
   - `grep` 查找内容、`管道 |` 组合命令的用法，让处理文本变得很方便。

## 三、遇到的挑战与解决
- **问题**：一开始总记不住 Git 命令，经常混淆 `reset` 和 `revert`。
- **解决**：把常用命令写在备忘录里，多练几次后慢慢就熟练了，还发现 `git --help` 查看帮助很有用。

## 四、未来计划
后续会继续练习分支管理和协作流程，争取在实际项目中用上这门课的知识。
