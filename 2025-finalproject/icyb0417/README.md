#这节课：Git、gihub使用及规范

    1.学习了git的工作区、暂存区、本地仓库以及远程仓库的作用，以及status仓库情况查询、增删查改、add暂存、commit提交、log日志b、reset等版本回退、diff比较操作。
    2.SSH Key的生成与在github远程仓库的使用。
    3.github仓库fork以及克隆仓库与pr推送拉取更新内容的远程仓库操作。
    4.查看远程仓库信息git remote -v
    5.分支的增checkout删new-branch查branch及合并且切换操作，其作用及意义。
    6.开发并提交功能git add .，git commit -m "实现用户登录功能"
    7.变基或合并到功能分支git checkout feature/user，git rebase main
